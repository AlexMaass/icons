## Rebuild fonts and cubing-icons.css

We use the excellent [gulp-iconfont](https://www.npmjs.com/package/gulp-iconfont).

- `npm install`
- `gulp` or `gulp watch`

## Deploy to GitHub Pages

- `gulp deploy`
